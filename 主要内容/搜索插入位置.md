# 搜索插入位置

```java
class Solution {
    public int searchInsert(int[] nums, int target) {
        int left = 0;
        int right = nums.length;
        while(left<right){
            int mid = left+(right-left)/2;
            if(nums[mid] == target){
                return mid;
            }
            if(nums[mid]>target){
                right = mid;
            }
            if(nums[mid]<target){
                left = mid+1;
            }
        }
        return left;
    }
}
```
最后没找到，要返回插入的位置，而插入的位置应该就是边界位置。即left或者right。
