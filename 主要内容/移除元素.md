# 数组中移除元素

```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int fast; 
        int slow = 0;
        for(fast = 0;fast<nums.length;fast++){
            if(nums[fast] != val){
                nums[slow] = nums[fast];
                slow++;
            }
        }
        //slow始终是指向新数组的空位置的，所以返回slow，就相当于返回新数组的size了
        return slow;
    }
}
```
